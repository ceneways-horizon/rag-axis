# Guides

This section covers practical, task-oriented guides for working with RAG Axis: implementing your own adapters, composing the three operational layers into a custom pipeline, deploying the server, using the launch development kit, and contributing to the project itself.

If you are looking for a complete, runnable pipeline instead, see [Examples](/docs/examples/basic-rag). If you are looking for conceptual background, see [Concepts](/docs/concepts/rag-system-layers).
